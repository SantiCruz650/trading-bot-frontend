Manual del Operador Antigravity

¿Qué hace el bot?

Antigravity es un motor de trading inteligente que utiliza modelos de Machine Learning para detectar regímenes de mercado y ejecutar estrategias de DCA (Dollar Cost Averaging) con gestión de riesgo institucional.

¿Cómo usarlo?

1. Inicia el bot con el botón START.
2. Monitorea la Exposición Global (ER).
3. Usa el Emergency Kill si detectas anomalías críticas.

Reglas Importantes

El sistema tiene un límite rígido de 20 niveles de DCA y un Hard Cap de riesgo del 30% (GEC). El bot dejará de comprar si se alcanzan estos límites para proteger tu capital.

Descargo de Riesgo

El trading de criptomonedas conlleva un riesgo significativo. El rendimiento pasado no garantiza resultados futuros. Usted acepta toda la responsabilidad por las pérdidas incurridas al usar este software.
